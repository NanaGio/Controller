## Sobre o Projeto
O Controller é um aplicativo desenvolvido para auxiliar confeiteiros e pequenos negócios na gestão de insumos.
A ideia é simplificar o controle de estoque, reduzir desperdícios e otimizar custos, permitindo que empreendedores foquem no que realmente importa: criar produtos deliciosos e de qualidade.

# Funcionalidades Principais
  - Gestão de Estoque: cadastro de insumos com quantidade, validade e custo.

  - Relatórios Simples: visão rápida de gastos, consumo e reposição necessária.

  - Alertas Inteligentes: notificações sobre insumos próximos em baixa quantidade.

# Ferramentas
  - Frontend: React.js
  - Backend: Node.js + Express
  - Banco de dados: Mongo DB
